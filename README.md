## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容彼此独立，不针对任何特定域名或网站，主要用于资源整理与历史存档目的。

## 目录说明

```
├── pages/          # 存放各 HTML 页面文件
├── assets/         # 页面引用的静态资源（CSS、JS、图片等，如有）
└── README.md       # 本文件
```

- **pages/**：包含所有归档的 HTML 页面，每个文件为一个独立页面。
- **assets/**：若页面需要外部资源（样式、脚本、媒体等），统一放置在此目录下，以便页面引用。
